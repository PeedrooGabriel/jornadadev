# Função ou Procedimento?

Str() -> Função. Retorna uma string como resultado da conversão de um tipo numérico;
QOut() -> Procedimento. Exibe texto na tela, mas não retorna um valor útil;
Date() -> Função. Retorna o valor da data atual do sistema. Este valor pode ser utilizado para "n" coisas no programa;
Len() -> Função, pois retorna o valor numérico que representa o tamanho de um array ou string. EX: "Batata" -> 6;
Upper() -> Função. Retorna a string modificada com todas as letras maiúsculas;
ClearScreen() -> Procedimento, pois executa uma atividade sem retorno útil. limpar a tela não retorna nada.
